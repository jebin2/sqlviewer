# SQLite Viewer

A browser-based SQLite database viewer and editor. Upload a `.sqlite` or `.db` file to browse, search, edit, and export.

**Live Demo:** [https://jebin2.github.io/sqlviewer/](https://jebin2.github.io/sqlviewer/)
